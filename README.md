# 🚀 ExamIQ  
### AI-Powered Past Paper Analyzer & Smart Study Planner

> Transforming past exam papers into actionable insights, predictions, and personalized study strategies.

---

## 📌 Problem Statement

Students rely heavily on past question papers but lack a structured way to:
- Identify high-frequency topics
- Understand question patterns & difficulty trends
- Align preparation with the official syllabus
- Optimize study time for maximum marks

Manual analysis is time-consuming and inefficient.

---

## 💡 Solution

**ExamIQ** is an AI-powered platform that:
- Analyzes past papers across multiple years
- Maps them to syllabus topics
- Predicts high-yield areas
- Generates a personalized, optimized study plan

---

## ✨ Key Features

### 📂 Multi-Paper Upload
- Upload multiple PDFs/images of past papers
- Supports multi-year & multi-subject inputs

### 🤖 AI Pattern Analysis
- Topic frequency detection  
- Question type classification  
- Difficulty trend analysis  
- Year-wise pattern insights  

### 📘 Syllabus Mapping
- Cross-references extracted topics with official syllabus  
- Highlights:
  - Covered topics  
  - Missing topics  
  - Low-frequency areas  

### 🎯 Topic Importance Scoring
- Ranks topics based on:
  - Frequency  
  - Recency  
  - Weightage trends  

### 🗓️ Smart Study Planner
- Auto-generates a personalized schedule  
- Prioritizes high-impact topics  
- Adapts to user goals  

### 📊 Visual Analytics Dashboard
- Topic frequency charts  
- Coverage heatmaps  
- Difficulty distribution graphs  

### 🧠 Practice Question Suggestions
- Recommends similar questions for important topics  
- Helps reinforce learning  

---

## 🚀 Advanced Features

### 🔮 Exam Question Predictor
- Predicts probable questions for upcoming exams  

### 🎯 Marks Maximizer Mode
- Input target score → get optimized study strategy  

### 🧠 Weak Area Detection
- Identifies gaps based on performance & coverage  

### 🧪 Mock Test Generator
- AI-generated tests based on past trends  

### 📝 AI Answer Evaluator
- Upload answers → get feedback & scoring  

### 📈 Exam Readiness Score
- Single metric representing preparation level  

### 🔗 Concept Dependency Graph
- Visualizes relationships between topics

## 🛠️ Tech Stack

**Frontend**
- React.js / Next.js  
- Tailwind CSS  
- Chart.js / Recharts  

**Backend**
- Node.js / FastAPI  
- REST APIs  

**AI / ML**
- NLP (topic extraction, classification)  
- LLM APIs (analysis, prediction, evaluation)  
- OCR (PDF/image parsing)  

**Database**
- MongoDB / PostgreSQL  

**Other Tools**
- Firebase / Supabase (auth & hosting)  
- Vercel / Netlify (deployment)  

---

## 🧩 System Workflow

1. Upload past papers  
2. Extract text using OCR  
3. Identify topics using NLP  
4. Map topics to syllabus  
5. Analyze trends & patterns  
6. Generate insights + predictions  
7. Create personalized study plan
